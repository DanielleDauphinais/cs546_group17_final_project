Welcome to Little Free Libary Connections: Hoboken

Where technology connects community book sharers.

Here at Little Free Libary Connections LFLC we want to connects users to Little free libraries in Hoboken.

On this website users can...

  • Create and interact with libraries

  • Add comments to a library

  • Rate the fullness of a library they visited

  • Find libraries on a map of Hoboken

  • Add libraries to your favorited list

The fullness rating feature is so donors know when the library needs to be refilled and book readers know when to visit the library
